# CRUD Web Application

This is a simple CRUD (Create, Read, Update, Delete) web application built using HTML, CSS, and JavaScript. The purpose of this project is to demonstrate the basic operations used to manage data in web applications.

## Project Description

CRUD stands for:

C - Create  
R - Read  
U - Update  
D - Delete  

These four operations allow users to interact with stored data.

This application allows users to:

- Add new data entries
- View existing records
- Update existing data
- Delete records

The project helps in understanding how frontend applications manage and manipulate data dynamically.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Add new records
- Display records in a table
- Edit existing data
- Delete records

## Learning Outcome

This project helps in understanding:

- DOM manipulation
- Event handling
- Basic CRUD operations
- Frontend data management
